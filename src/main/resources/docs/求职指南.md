# 程序员求职指南

## 一、求职准备阶段

### 1.1 自我定位
**明确求职目标：**
- 技术方向：后端开发、前端开发、全栈、大数据、AI等
- 行业选择：互联网、金融科技、电商、传统企业数字化等
- 公司规模：大厂、中厂、创业公司、外企
- 薪资期望：根据市场行情和自身能力合理定位

**评估自身能力：**
- 技术栈掌握程度
- 项目经验积累
- 学历背景优势
- 软技能水平（沟通、协作、学习能力）

### 1.2 简历准备
**简历结构：**
```
1. 个人信息（姓名、电话、邮箱、GitHub/博客链接）
2. 求职意向（目标岗位、期望城市、薪资范围）
3. 教育背景（学校、专业、学历、时间）
4. 专业技能（技术栈、工具、语言）
5. 工作经历/实习经历（公司、岗位、时间、职责、成果）
6. 项目经历（项目名、角色、技术栈、职责、成果、难点）
7. 其他（证书、获奖、开源贡献、技术博客）
```

**简历优化技巧：**
- **量化成果**：使用数据说话（如"优化后系统性能提升30%"）
- **STAR法则**：情境(Situation)、任务(Task)、行动(Action)、结果(Result)
- **关键词匹配**：根据JD调整简历关键词
- **一页原则**：应届生/1-3年经验尽量控制在一页
- **PDF格式**：避免Word格式排版错乱

**避免的坑：**
- 不要写与岗位无关的内容
- 不要夸大或造假
- 不要出现错别字
- 不要贴生活照片

## 三、面试流程

### 3.1 面试类型
| 类型 | 形式 | 考察重点 |
|------|------|----------|
| 电话/视频初筛 | 15-30分钟 | 基本情况、沟通表达 |
| 技术一面 | 45-60分钟 | 基础知识、编码能力 |
| 技术二面 | 45-60分钟 | 项目经验、技术深度 |
| 技术三面/架构面 | 60分钟+ | 系统设计、架构能力 |
| HR面 | 30-45分钟 | 价值观、稳定性、薪资 |
| 总监/VP面 | 30分钟 | 综合素质、潜力 |

### 3.2 面试准备清单
**技术准备：**
- [ ] 基础知识复习（语言、框架、算法）
- [ ] 项目梳理（准备3-5个项目的详细介绍）
- [ ] 算法刷题（LeetCode 200+）
- [ ] 系统设计学习
- [ ] 公司背景调研

**材料准备：**
- [ ] 简历打印版（多份）
- [ ] 身份证
- [ ] 学历证书/成绩单（应届生）
- [ ] 作品集/GitHub链接
- [ ] 笔记本（记录面试问题）

**形象准备：**
- [ ] 着装得体（商务休闲）
- [ ] 提前到达（提前10-15分钟）
- [ ] 保持手机静音

## 四、薪资谈判

### 4.1 薪资构成
```
总包 = 月薪 × 月数 + 年终奖 + 股票/期权 + 签字费 + 其他福利
```

**常见福利：**
- 五险一金缴纳基数和比例
- 补充医疗保险
- 餐补/交通补/通讯补
- 带薪年假
- 弹性工作/远程办公
- 培训学习基金

### 4.2 谈薪技巧
**时机选择：**
- 不要在初筛时谈薪
- 等拿到正式offer或HR明确询问时再谈
- 有多个offer时谈判空间更大

**策略建议：**
1. **调研市场**：了解目标岗位的市场薪资范围
2. **锚定效应**：先让对方出价，或给出一个略高于预期的范围
3. **整体考虑**：不仅看月薪，还要看总包和长期发展
4. **留有余地**：不要一口回绝，可以表示"需要考虑"

**话术示例：**
```
"根据我的调研，这个岗位的市场薪资范围是XX-XX。
考虑到我的经验和技能，期望薪资是XX。
当然，我也看重公司的发展平台和团队氛围，
希望能有一个双方都满意的方案。"
```

## 五、Offer选择

### 5.1 评估维度
| 维度 | 权重 | 评估要点 |
|------|------|----------|
| 薪资福利 | 25% | 总包、涨幅、福利 |
| 发展空间 | 25% | 晋升通道、技术成长 |
| 业务前景 | 20% | 行业趋势、公司发展 |
| 团队氛围 | 15% | 领导风格、同事关系 |
| 工作生活 | 15% | 加班情况、通勤距离 |

### 5.2 避坑指南
**警惕信号：**
- 薪资结构复杂，基本工资低
- 试用期过长（超过6个月）
- 五险一金按最低基数缴纳
- 合同与offer不一致
- 公司负面新闻多

**确认事项：**
- 入职时间
- 试用期时长和薪资
- 转正标准
- 离职流程和竞业协议

## 六、不同阶段的求职策略

### 6.1 应届生
**时间规划：**
- 大三暑假：实习
- 大四上学期：秋招（7-11月）
- 大四下学期：春招（3-5月）

**重点准备：**
- 算法基础（LeetCode 150+）
- 计算机基础（操作系统、网络、数据库）
- 实习经历（至少1-2段）
- 项目经验（课程项目、开源项目）

### 6.2 1-3年经验
**发展方向：**
- 深耕技术：成为某个领域的专家
- 拓展广度：全栈或架构方向
- 转向管理：技术管理路线

**求职重点：**
- 项目成果和影响力
- 技术深度和广度
- 解决复杂问题的能力

### 6.3 3-5年经验
**职业选择：**
- 大厂P6/P7：技术专家路线
- 中小厂技术负责人：带团队
- 创业公司CTO：技术合伙人

**核心竞争力：**
- 架构设计能力
- 技术选型能力
- 团队管理经验
- 业务理解能力

### 6.4 5年以上经验
**发展方向：**
- 架构师/技术专家
- 技术总监/VP
- 技术创业
- 独立顾问

## 七、求职心态管理

### 7.1 常见挑战
- **简历石沉大海**：海投无反馈
- **面试挂掉**：技术面或HR面不通过
- **薪资不满意**：offer薪资低于预期
- **选择困难**：多个offer不知如何选

### 7.2 心态调整
1. **接受拒绝**：面试挂掉是常态，总结经验继续前进
2. **保持学习**：求职期间也要持续学习
3. **适当休息**：避免过度疲劳和焦虑
4. **寻求支持**：和朋友、前辈交流

### 7.3 时间管理
- 每天投递5-10份简历
- 每周安排2-3场面试
- 预留复习和总结时间
- 保持规律作息

## 八、入职准备

### 8.1 入职前
- 完成离职手续（如有）
- 准备入职材料（身份证、学历证、银行卡等）
- 了解公司文化和团队
- 学习相关技术栈

### 8.2 入职后
- 快速熟悉业务和技术架构
- 主动与同事建立关系
- 设定试用期目标
- 定期与mentor/主管沟通
